# NLP Web App

This is a Natural Language Processing (NLP) Web App built using Streamlit. It leverages pre-trained transformers and spaCy to perform various NLP tasks.

## Features
- **Text Summarization**
- **Named Entity Recognition**
- **Sentiment Analysis**
- **Question Answering**
- **Text Completion**
